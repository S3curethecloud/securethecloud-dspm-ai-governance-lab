# Governance Boundaries

## Lab Boundary

This repository is a synthetic DSPM and AI governance simulation lab. It is intentionally designed to be safe for portfolio demonstration, interview discussion, architecture review, and control design walkthroughs.

## Allowed

- Synthetic sensitive data fixtures.
- Deterministic posture scoring.
- Policy recommendation simulation.
- Evidence record generation.
- Local API and dashboard development.
- Microsoft Purview and Copilot conceptual mapping.
- Cross-reference to companion SecureTheCloud Microsoft Copilot Governance Lab.

## Blocked

- Real Microsoft 365 data.
- Real Purview tenant data.
- Microsoft Graph write operations.
- Tenant-wide permission grants.
- Production DLP mutation.
- Production Insider Risk Management mutation.
- Automated enforcement.
- Autonomous approval.
- Real user monitoring.

## Review Rule

Every future phase must prove the boundary is preserved before moving forward.
